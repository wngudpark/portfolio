## 프로젝트 개요

외교부 해외안전상식 홍보를 위한 이벤트성 모바일 웹 퀴즈 게임입니다. 참여자가 닉네임을 입력한 뒤 90초 안에 30개의 안전상식 문제를 풀고, 획득 점수에 따라 '우승 / 준우승 / 4강 / 8강 / 16강 / 조별리그' 랭킹을 부여받습니다. 8강 이상(160점) 달성자에게는 개인정보 동의 기반의 경품 응모 기능을 제공하여, 단순 퀴즈를 넘어 참여자 정보 수집과 경품 추첨까지 이어지는 이벤트 전체 흐름을 구현했습니다.

## 담당 업무

- **화면 전환형 SPA 설계**: 별도 라우팅 라이브러리 없이 React 상태 관리 기반으로 닉네임 입력, 규칙 안내, 퀴즈, 결과, 응모까지 5개 화면을 전환하고, 닉네임·점수 같은 공통 게임 상태를 상위 컴포넌트에서 관리하도록 구성했습니다.
- **퀴즈 게임 로직 구현**: 서버에서 무작위로 30문항을 받아와 문항당 10점(최대 300점)으로 채점하고, 정답·오답 피드백을 보여준 뒤 자동으로 다음 문제로 넘어가도록 구현했습니다.
- **제한시간·이탈 방지 처리**: 90초 타이머 바를 구현하고, 일정 시간 무동작 시 자동으로 초기 화면으로 복귀시키는 idle 타이머를 적용해 이벤트 환경에서의 방치 세션을 방지했습니다.
- **결과·랭킹 화면**: 점수 구간별로 랭킹과 이미지, 맞춤 메시지를 매핑하고, 점수 카운트업 애니메이션과 클립보드 링크 공유·토스트 알림 기능을 구현했습니다.
- **경품 응모 플로우**: 개인정보 수집·이용 동의 모달, 연락처 자동 포맷팅 및 유효성 검증, 8강 이상 달성자에게만 응모 버튼이 노출되는 조건부 게이팅을 구현했습니다.
- **PHP API 개발**: 문항 제공, 게임 완료 기록, 경품 응모 처리 엔드포인트를 구현하고 원격 MySQL과 연동했습니다.
- **중복 응모 방지**: 응모 데이터를 저장하기 전 연락처를 기준으로 기존 레코드를 조회하고, 중복된 경우 적절한 오류 응답과 함께 안내 메시지를 반환하도록 처리했습니다.
- **SQL 인젝션 대응**: 사용자 입력이 들어가는 응모·완료 저장 로직에 Prepared Statement를 적용해 입력값을 파라미터 바인딩 방식으로 안전하게 처리했습니다.
- **입력값 정제·검증**: 닉네임은 정규식 화이트리스트(한글·영문·숫자)로 필터링한 뒤 글자 수를 제한했고, 연락처는 숫자만 추출해 자릿수 형식을 프론트엔드와 백엔드 양쪽에서 검증했습니다.
- **완료 기록 비동기 분리**: 게임 완료 기록 API는 실패하더라도 사용자 경험에 영향을 주지 않도록 비동기로 분리 처리해, 네트워크 오류가 게임 진행을 끊지 않도록 구성했습니다.
- **API 실패 대비 UI**: 문항 로딩 실패 시 재시도 안내 화면을 노출해, 빈 화면 노출 없이 사용자가 상황을 인지하고 대응할 수 있도록 처리했습니다.
