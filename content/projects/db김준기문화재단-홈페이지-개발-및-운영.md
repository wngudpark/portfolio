## 프로젝트 개요

DB김준기문화재단 공식 사용자 포털입니다. 재단소개·주요사업(드림빅/드림리더/드림마스터 등)·재단소식을 제공하고, 회원가입·NICE 본인인증·장학금 신청·공모전 신청·마이페이지·커뮤니티(DB커넥트·라운지·드림보드)를 포함합니다. 영문 페이지도 별도 컨트롤러로 운영하며, 관리자 포털·IFC 사이트와 동일한 데이터베이스를 공유합니다.

## 담당 업무

- **장학금 신청**: 드림리더·드림마스터·드림서포트·드림파이낸스 유형별 신청 폼을 구현했습니다. 코드성 데이터는 캐싱 처리했고, 업로드 파일은 임시 저장 후 확정 시 이동하는 방식으로 처리했습니다.
- **공모전 신청**: 공모전 접수 로직을 별도 패키지로 분리해 구현했습니다.
- **마이페이지**: 신청 현황·수혜 이력·추진이유서 조회 및 증명서 발급 신청을 처리했습니다. 발급 시 담당자 메일을 비동기로 발송했습니다.
- **증명서 PDF 생성**: HTML을 PDF로 변환하는 라이브러리를 사용해, 한글 폰트를 임베딩한 증명서를 생성했습니다.
- **회원·인증**: NICE 휴대폰·아이핀 본인인증을 연동(토큰 생성 → 암호화 데이터 전달 → 콜백 복호화)하고, 비밀번호 90일 만료·유예 정책을 구현했습니다.
- **커뮤니티**: 팔로우·블랙리스트·인맥 기능, 드림보드, 게시판·댓글 기능을 구현했습니다. 삭제·알람 읽음 처리는 소유권 검증 후 수행하도록 했습니다.
- **방문자 통계 필터**: 비정적 요청을 재단/공모전으로 구분해 방문 기록을 저장했습니다.
- **인프라 구성**: DB Inc.에서 제공받은 Windows 서버 환경에서 WEB, WAS, DB 서버 구성을 직접 설계하고 구축했습니다.

## 개선 내용

- **비밀번호 암호화 방식 무중단 전환**: 로그인 시 기존 방식과 신규 방식을 함께 검증하도록 하고, 로그인 성공 시 레거시 비밀번호를 즉시 새 방식으로 업그레이드해 운영 중단 없이 전환했습니다.
- **하드코딩 시크릿 제거**: 암호화 키·IV 리터럴을 프로퍼티 주입 방식으로 이관하고, 이메일 인증 코드 난수 생성기를 보안 강화된 방식으로 교체했습니다.
- **파일 업로드 보안 강화**: 파일 앞부분의 매직 바이트를 검증해 확장자 위변조를 차단하고, 파일명을 정제해 경로 조작 공격을 방지했습니다.
- **IDOR 취약점 차단**: 증명서 다운로드 시 소유권을 DB에서 확인하고, 경로를 2차로 정규화 검증해 불일치 시 접근을 차단했습니다.
- **DataSource 교체**: 요청마다 커넥션을 새로 생성하던 방식에서 커넥션 풀 기반 방식으로 교체해 연결 지연을 줄였습니다.
- **캐시 도입**: 메뉴·장학 옵션 조회를 인메모리 캐시로 전환해 매 요청마다 발생하던 DB 조회를 줄이고, 관리자 권한으로 수동 초기화할 수 있는 기능을 추가했습니다.
- **NICE 인증 로직 서비스 분리**: 컨트롤러에 인라인으로 작성되어 있던 복호화 로직을 별도 서비스 메서드로 추출해 가독성과 재사용성을 높였습니다.
- **XSS 필터 오탐 수정**: NICE 인증 콜백 파라미터가 XSS 패턴에 오탐되어 콜백이 차단되던 문제를, 콜백 경로·파라미터명 예외 처리로 해결했습니다.
