## 프로젝트 개요

서울디자인재단의 의뢰로 개발한, 파리 Maison & Objet 박람회에 참가하는 한국 디자인 브랜드를 해외 바이어에게 소개하기 위한 B2B 쇼케이스 웹서비스입니다. 브랜드를 카테고리(Living / Traditional / Sustainability)별로 소개하고, 바이어가 상품 페이지에서 옵션·수량을 선택하면 수량 구간별 단가와 부가세(10%)가 반영된 간이 견적서를 생성해 이미지로 다운로드할 수 있습니다. 프론트엔드(React), 데이터 API(PHP/MySQL), 브랜드사가 직접 콘텐츠를 관리하는 관리자 CMS까지 프론트 퍼블을 제외한 전 과정을 혼자 전담 개발했습니다.

## 담당 업무

- **프론트엔드 SPA 전체 구축**: React, TypeScript 기반으로 메인, 브랜드 상세, 상품 상세, 견적서, 약관 등 8개 라우트를 구성했습니다.
- **상품 견적 로직**: 옵션 선택과 수량 구간(티어)별 단가 계산, 실시간 합계·부가세 산출, 유럽식(EUR) 통화 표기 처리를 구현했습니다.
- **견적서 생성/다운로드**: 선택 내역을 견적서 화면으로 전달하고, html2canvas로 화면을 PNG 이미지로 캡처해 다운로드하는 기능을 구현했습니다.
- **PHP 데이터 API 5종 개발**: 브랜드 목록, 브랜드 상세, 상품 등 MySQL 다중 테이블 조인 결과를 옵션·가격 티어·이미지 단위로 가공해 응답하는 API를 구현했습니다.
- **관리자 CMS 개발**: 브랜드·상품·옵션·할인 구간·디자이너 정보 CRUD와 대표·상세 이미지 업로드 기능을 구현했습니다.
- **배포 구성**: Apache 설정으로 SPA 라우팅 폴백과 관리자 경로 분리를 처리하고, 프로덕션·개발 환경별로 API 경로를 분기했습니다. 박람회 현장(파리)에서의 접속 지연 문제 해결을 위해 AWS 인프라로 서버를 이관했습니다.

## 문제 해결

- **해외 접속 지연 해결을 위한 인프라 개선**: 초기에는 국내 서버로 배포했으나, 실제 박람회가 열리는 파리 현지에서 화면 로딩이 눈에 띄게 느린 문제가 발생했습니다. 원인이 물리적 거리로 인한 네트워크 지연이라는 것을 파악하고 서버 리전을 파리로 이관했지만, 리전 이관만으로는 원하는 수준까지 속도가 개선되지 않았습니다. AWS Global Accelerator를 추가로 적용해 AWS 백본 네트워크를 통한 경로 최적화를 더하면서, 현지 접속 속도를 목표 수준까지 끌어올릴 수 있었습니다.
- **이미지 깜빡임·레이아웃 흔들림 방지**: 데이터 로드 시 관련 이미지를 미리 로드(preload)하도록 처리하고, 로드 실패 시에도 렌더링이 멈추지 않도록 예외를 방어적으로 처리했습니다.
