## 프로젝트 개요

WWF Korea(한국세계자연기금)의 꿀벌·자연보전 캠페인 참여를 유도하기 위한 미니게임입니다. 사용자가 닉네임을 입력하고 튜토리얼을 거쳐 게임을 플레이한 뒤, 점수와 전체 랭킹을 확인하고 뉴스레터 구독·후원 페이지로 연결되는 흐름으로 구성됩니다. 웹/모바일 버전과, 오프라인 전시·현장 부스에서 무인으로 운영되는 키오스크(Electron) 버전 두 가지로 개발했습니다.

## 담당 업무

### 웹/모바일 버전

- **게임 클라이언트**: 3개 레인을 오가며 꿀을 획득하고 말벌을 피하는 러너형 게임을 구현했습니다. 생명 3개, 시간 경과에 따른 점수 획득, 키보드·터치 스와이프를 지원하는 상하 이동 조작을 구현했습니다.
- **스테이지 연출**: 일정 주기로 배경(꽃밭→사막→비)이 전환되고, 스테이지가 올라갈수록 장애물 속도와 생성 간격이 증가하도록 구성했습니다. 스테이지 전환 시 오버레이·파티클 효과를 적용해 난이도 상승을 시각적으로 연출했습니다.
- **결과·공유 페이지**: 개인 순위와 전체 상위 랭킹을 산출해 보여주고, 링크 공유와 WWF 후원·뉴스레터 연동 기능을 구현했습니다. 이벤트 응모 폼에는 개인정보 동의, 중복 응모 검증 로직을 포함했습니다.
- **관리자 백오피스**: JWT 기반 쿠키 인증으로 로그인 기능을 구현하고, 기간별 참여자 검색, 일자별/이벤트별 참여자 조회 및 페이징, 통계 집계, 엑셀 다운로드 기능을 제공했습니다.

### 키오스크 버전 (Electron)

- **키오스크 셸 구성**: 전체화면·키오스크 모드로 항상 최상단에 표시되도록 구성하고, 개발 환경과 프로덕션 환경을 분기해 개발 시에만 개발자 도구를 허용하도록 했습니다.
- **키오스크 잠금**: 시스템 단축키와 새 창, 우클릭 메뉴, 개발자 도구 접근을 차단하고, 부팅 시 앱이 자동 실행되도록 구성해 무인 운영 환경을 만들었습니다.
- **운영자용 종료 동선**: 게임 플레이 화면을 제외한 특정 화면 영역을 짧은 시간 내 연속 클릭하면 앱이 종료되는 히든 제스처를 별도 통신 채널을 통해 안전하게 구현했습니다.
- **무인 자동 복귀**: 일정 시간 조작이 없으면 홈 화면으로 복귀하고, 결과 화면은 카운트다운 후 자동으로 초기화되도록 구현했습니다. 점수는 로컬 저장소에 저장해 결과 화면에 표시했습니다.
- **오프라인 이식**: 서버·DB에 의존하던 로직(닉네임, 랭킹, 이벤트 응모 등)을 제거하고 정적 리소스 기반으로 재구성했으며, Windows 설치 파일로 빌드해 배포했습니다.
